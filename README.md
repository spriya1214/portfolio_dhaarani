# Dhaarani Nambi - DevOps Portfolio

A modern, responsive portfolio website for Dhaarani Nambi, an aspiring DevOps Engineer passionate about automation and cloud technologies.

## Features

- 🎨 Modern UI with beautiful animations
- 📱 Fully responsive design
- 🌙 Dark mode support
- ⚡ Fast performance with Vite
- 🎯 Clean, maintainable code structure

## Tech Stack

- **Frontend**: React 18 with TypeScript
- **Styling**: TailwindCSS + custom components
- **Animations**: Framer Motion
- **Build Tool**: Vite
- **Icons**: Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5175`

## Building for Production

```bash
npm run build
```

## Deployment

This project is configured for automatic deployment to GitHub Pages:

```bash
npm run deploy
```
